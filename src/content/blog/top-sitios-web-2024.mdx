Mi selección personal de sitios web awesómicos. Aquí encontrarás una lista cuidadosamente curada de páginas que destacan por su diseño, funcionalidad o creatividad.

- ### musicForProgramming();

  Serie de listas musicales pensadas para escuchar mientras programas, ayudando a enfocar la mente e inspirar la creatividad. Además, visualmente es muy agradable.  
  [https://musicforprogramming.net](https://musicforprogramming.net/?ref=baumannzone.dev)

- ### Multiple Window 3D Scene

  Abre este sitio web en 2 o más ventanas y observa cómo se sincronizan las animaciones en todas ellas. Creado con Three.js.  
  [https://bgstaal.github.io/multipleWindow3dScene](https://bgstaal.github.io/multipleWindow3dScene/?ref=baumannzone.dev)

- ### Creatica

  Mueve el cursor en este sitio web y mira que efecto tan genial se produce.
  [https://creatica.app](https://creatica.app/?ref=baumannzone.dev)

- ### WebCheck

  Es una herramienta OSINT y de seguridad todo en uno, para obtener datos de cualquier sitio web. Además, el sitio web se ve fenomenal.  
  [https://web-check.xyz](https://web-check.xyz/?ref=baumannzone.dev)

- ### Wanderboat AI

Sitio web para planear viajes con una IA. El efecto de parallax y las animaciondes del scroll están muy bien logradas.  
[https://wanderboat.ai](https://wanderboat.ai/?ref=baumannzone.dev)

- ### WebAmp

WinAmp 2, el clásico reproductor de música de los 90s ha vuelto, pero esta vez como un sitio web. Programado con React.  
[https://webamp.org](https://webamp.org/?ref=baumannzone.dev)

- ### Sacred OS

Una web que es sistema operativo inspirado en Windows 9x y está escrito en JavaScript. Es interactivo y hasta puedes juegar al tetris.  
[https://sacred.neocities.org/](https://sacred.neocities.org/?ref=baumannzone.dev)

- ### Flighty

Flighty es una app de seguimiento de vuelos en tiempo real que ofrece actualizaciones instantáneas, predicción de retrasos y alertas sobre cambios en vuelos. Su web es muy atractiva visualmente.  
[https://flighty.com/](https://flighty.com/?ref=baumannzone.dev)

- ### superpower

Superpower es una plataforma de salud que ofrece suscripciones para quienes buscan optimizar su bienestar. Su web está muy bien diseñada y es un ejemplo claro de cómo el minimalismo y las animaciones sutiles pueden hacer que una web se vea moderna y profesional.  
[https://superpower.com/](https://superpower.com/?ref=baumannzone.dev)
